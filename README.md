# VisionTrack Sports

VisionTrack Sports is a collection of AI-powered sports computer vision projects built for learning, experimentation, and real-time sports analytics using modern deep learning and computer vision techniques.

The project demonstrates player detection, object tracking, team classification, heatmap generation, ball/puck tracking, and match analytics from broadcast sports footage across multiple sports.

# Projects
# 🏀 Basketball Match Analytics

A real-time basketball analytics system for tracking players, ball possession, movement patterns, and team performance.

Features

Player Detection
Ball Detection
Multi-Object Tracking
Team Classification
Ball Possession Analysis
Camera Motion Compensation
Team Heatmaps
Activity Analytics
Interactive Match Dashboard

# Run

python basketball_match_analytics.py
